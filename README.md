# Barista Café Website

A responsive Barista Café website built using **HTML, CSS, and Bootstrap**. This project focuses on creating a modern, clean, and user-friendly interface suitable for a café or coffee shop business, with an emphasis on responsive design and structured layout.

## Tech Stack
- HTML5  
- CSS3  
- Bootstrap 5  


## Features
- Fully responsive design for mobile, tablet, and desktop devices  
- Clean and modern user interface  
- Bootstrap grid system for consistent layout  
- Structured sections for café information and menu highlights  
- User-friendly navigation and readable content
